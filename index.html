SDL 2 Examples

Collection of small examples written in C++ and using SDL 2.
No executable files are provided for now, but it is possible to build these examples using CMake from sources (see links below).
These examples were tested on Windows and Linux desktop operating systems.

Note for building on Windows: you may need to add sdl2-config.cmake to your directory with SDL 2 library to build the examples.
