Wolord

Genre(s) Puzzle
Mode(s) Single Player
Initial Release(s) Prototype, May 9, 2022 (Windows), June 20, 2022 (macOS)
Last Release(s) Beta, 0.9.2, April 25, 2023 (Windows)
Platform(s) Windows, macOS

Wolord is a minimalistic endless word puzzle game.

Game on Indie DB | itch.io | Game Jolt

Windows XP is not supported.

Important Notes for Windows Vista/7/8.0/8.1:

  • Update for Universal C Runtime (KB2999226 Windows Update) must be installed to run the game.
  • Install Microsoft Visual C++ Redistributable 2015-2022** if you have the following (or similar) error in the Setup (Installer): "An error occurred while installing system components for ...". After this, you should be able to run Setup (Installer) again without the error.
**Make sure to select correct version(s), x86 and/or x64
